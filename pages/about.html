welcome to Coffee Project!

This project is created to practice my web programming skills which were learned explicitly through the WWW.

This webpage in NOT created for a real coffee shop, it was made for fun as a hobby.

Here you can see a website, witch is responsive and has an adaptive design.
It has the option to create a table reservation through a form that allows you to reserve in a week from now and up to 3 months forward. After sending the form it will save the time and date to the local storage for display. There is also an option to clear all or one of the reservations made through this webpage from the client side view ('reserved' menu).

All of the menu Item are dynamically generated from js objects

It has an 'Events' scroll menu, which scrolls one element at a time.

Currently you can see here some ready stuff and some uncompleted functionality.
